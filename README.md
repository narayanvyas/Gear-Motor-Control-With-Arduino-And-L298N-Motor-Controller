# Arduino Gear Motor Control With L298N Motor Controller And IR Remote
Gear Motor Control With L298N Motor Controller, IR Remote And Arduino.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. L298N Motor Driver
4. 2 x Gear Motors
5. Jumper Wires
6. IR Receiver And Remote
7. Battery Pack
